Hexo 建站指南

发表于 2019-02-28 | 更新于 2019-04-02 | 分类于 hexo

如何安装， 配置你的hexo。

Prerequisites

1. Install node.js

2. Install HEXO

Getting Started

  1. 建立your_name.github.io 仓库
  2. 将仓库clone下来

  3. 初始化

  4. 浏览器打开[http://localhost:4000/]

Usage

  • hexo generage (hexo g)
  • hexo server (hexo s)
  • hexo deploy (hexo d)
  • hexo new (hexo n)
    • hexo new “postName” #新建文章
    • hexo new page “pageName” #新建页面

In combinaton:

  • hexo d -g #生成部署
  • hexo s -g #生成预览
